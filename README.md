# Logistic Regression From Scratch

## Overview

This project implements **Logistic Regression from scratch** using **NumPy** without using `scikit-learn`'s LogisticRegression class.

## Features

* Logistic Regression implementation from scratch
* Sigmoid Activation Function
* Binary Cross-Entropy Loss
* Gradient Descent Optimization
* Prediction on new data
* Accuracy Evaluation

## Technologies

* Python
* NumPy
* Google Colab

## How it Works

1. Initialize weights and bias.
2. Compute the linear equation.
3. Apply the Sigmoid function.
4. Calculate Binary Cross-Entropy Loss.
5. Update weights and bias using Gradient Descent.
6. Repeat for a fixed number of iterations.
7. Make predictions on unseen data.
